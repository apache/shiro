Apache Shiro + Spring Web + Hibernate Example
=============================================

A Spring Boot example web application that show the usage of a user login with Spring and Hibernate.

Run the Example
---------------

```
mvn jetty:run
```

Browse to `http://localhost:9080/`

Or deploy to Heroku:
